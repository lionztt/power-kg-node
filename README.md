# 皮皮电力运检知识图谱分析系统后端

项目路径：
- bin 项目服务配置
- conf 项目其他配置
- controller 控制器层
- model model层
- public 公共文件
- routes 项目路径
- service 项目服务
- view 项目模板
- utils 工具包
    - http.js 异步请求配置文件
- app.js 应用入口文件
- .gitignore git忽略文件
- package.json 项目依赖文件
- package-lock.json 项目依赖版本锁文件


项目启动方法：
1. 进入项目文件夹
2. npm i 安装项目依赖
3. npm run dev 启动项目
4. 浏览器访问 http://localhost:3033/
